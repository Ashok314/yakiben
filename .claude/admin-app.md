Here’s a summary of each page in admin :

login
- role based login 

Orders:

Displays a list of all orders.
Includes search, filter, and sort functionality.
Allows viewing order details and updating order statuses (e.g., preparing, ready, delivered).
Accessible to all roles (staff, manager, delivery).

Menu Management:

Allows managers to manage the restaurant's menu.
Features include adding, editing, and deleting menu items.
Supports uploading images, setting prices, and categorizing items.

Delivery:

Displays a list of delivery orders.
Includes driver assignment and tracking.
Accessible to managers and delivery personnel.

User Management:

Allows managers to manage user accounts.
Features include adding, editing, and removing users.
Supports assigning roles (e.g., staff, manager, delivery).

Settings:

Provides restaurant-specific settings:
Restaurant Info: Update name, address, and contact details.
Banner: Manage promotional banners or announcements.
Printer Settings: Configure printers for auto-printing order receipts.
